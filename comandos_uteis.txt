GERAR CHAVE SECRETA DO JWT (JWT SECRET KEY 256-BIT BASE64)

Opção 1: Usando OpenSSL (Recomendado - Git Bash/Linux)
openssl rand -base64 32

Opção 2: Usando Python
python -c "import secrets, base64; print(base64.b64encode(secrets.token_bytes(32)).decode())"

Opção 3: Usando Node.js
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"


GERAR SENHA FORTE PARA O BANCO DE DADOS

Opção 1: Usando OpenSSL
openssl rand -base64 16

Opção 2: Usando Python (Senha puramente alfanumérica + símbolos seguros)
python -c "import secrets, string; print(''.join(secrets.choice(string.ascii_letters + string.digits + '-_!?') for i in range(20)))"

TESTES DE API (Windows CMD / PowerShell)
Lembre-se de substituir SEU_TOKEN_AQUI pelo token JWT válido.
Utilizamos curl.exe e aspas duplas (\") para rodar nativamente no Windows.


1) Listar Produtos (GET)
curl.exe -i --location "http://localhost:8090/product" ^
--header "Authorization: Bearer SEU_TOKEN_AQUI"

2) Criar Produto (POST)
curl.exe -i --location --request POST "http://localhost:8090/product" ^
--header "Content-Type: application/json" ^
--header "Authorization: Bearer SEU_TOKEN_AQUI" ^
--data "{\"name\": \"Teclado Gamer\", \"description\": \"RGB Mecânico\", \"price\": 350.0, \"category\": \"Periféricos\", \"stock\": 20}"

3) Atualizar Produto (PUT)
curl.exe -i --location --request PUT "http://localhost:8090/product" ^
--header "Content-Type: application/json" ^
--header "Authorization: Bearer SEU_TOKEN_AQUI" ^
--data "{\"id\": 2, \"name\": \"Notebook Gamer (Atualizado)\", \"description\": \"RTX 4090, 64GB RAM\", \"price\": 16500.0, \"category\": \"Eletrônicos\", \"stock\": 10}"

4) Realizar Compra (POST)
curl.exe -i --location --request POST "http://localhost:8090/purchase" ^
--header "Content-Type: application/json" ^
--header "Authorization: Bearer SEU_TOKEN_AQUI" ^
--data "{\"clientId\": 1, \"productId\": 2, \"quantity\": 2}"
