GERAR CHAVE SECRETA DO JWT (JWT SECRET KEY 256-BIT BASE64)

Opção 1: Usando OpenSSL (Recomendado - Git Bash/Linux/macOS)
openssl rand -base64 32

Opção 2: Usando Python
python -c "import secrets, base64; print(base64.b64encode(secrets.token_bytes(32)).decode())"

Opção 3: Usando Node.js
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"


GERAR SENHA FORTE PARA O BANCO DE DADOS

Opção 1: Usando OpenSSL
openssl rand -base64 16

Opção 2: Usando Python (Senha alfanumérica + símbolos seguros)
python -c "import secrets, string; print(''.join(secrets.choice(string.ascii_letters + string.digits + '-_!?') for i in range(20)))"


DOCKER COMPOSE - MODOS DE EXECUÇÃO

1) Modo Desenvolvimento (Local com Spring DevTools e Live Reload):
docker compose -f docker-compose.yml -f docker-compose-dev.yml up --build -d

2) Modo Watch (Sincronização de arquivos em tempo real no contêiner - Recomendado):
docker compose -f docker-compose.yml -f docker-compose-dev.yml watch
ou
docker compose -f docker-compose.yml -f docker-compose-dev.yml up --watch

3) Modo Produção (Build final autocontido):
docker compose up --build -d

4) Parar e remover todos os contêineres:
docker compose -f docker-compose.yml -f docker-compose-dev.yml down

5) Remover todas as imagens do Docker
docker rmi -f $(docker images -q)
ou 
docker rm -f ollama mysql_db spring_app ollama_pull


GERENCIAMENTO DO MYSQL VIA DOCKER

Acessar o terminal interativo do MySQL dentro do contêiner:
docker exec -it mysql_db mysql -u user -p"kwd2%/O.?NSN4ZwWXJ[Z" rest_db

Ver as tabelas existentes:
docker exec -i mysql_db mysql -u user -p"kwd2%/O.?NSN4ZwWXJ[Z" rest_db -e "SHOW TABLES;"

Visualizar todos os produtos:
docker exec -i mysql_db mysql -u user -p"kwd2%/O.?NSN4ZwWXJ[Z" rest_db -e "SELECT * FROM products;"
